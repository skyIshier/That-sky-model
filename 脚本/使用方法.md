依赖关系

·Python 3.6+（标准库：ctypes、struct、io、os、sys、Globe、argparse、re） ·LZ4库-用于解压缩。

1安装python 
可以到官网下载  
Termux:
```
# 更新包管理器

pkg update && pkg upgrade

# 安装Python
pkg install python
```
2安装Lz4库
Termux:
```
# 使用pip安装
pip install lz4

# 如果上面的命令失败，尝试
pkg install python-lz4
pip install lz4
```
3文件使用方法
1.保证py文件和模型文件在同一个目录下面变形后可以选择换哪一个文件可以多选
```
python py文件名.py
```
2.
也是保证在同一目录下
```
python py文件名.py 模型文件名.mesh
```
