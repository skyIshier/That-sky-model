《光遇》模型与APK修改工具集完全使用手册

================================================================================
前言
================================================================================

本工具集专为《光遇》游戏资源解析与修改设计，由多位作者联合开发并免费开源。
请尊重作者劳动成果，严禁倒卖。

工具集主要包含两大核心模块：

【模块一】模型转换工具
将游戏中的 .mesh 文件转换为通用 .obj 格式，支持导入 Blender、Prisma3D 等
主流建模软件，同时支持将修改后的 .obj 文件重新打包回 .mesh 格式。

【模块二】APK修改工具
直接修改游戏安装包（APK文件），实现全物品解锁、完整性检查绕过及地图功能修改，
无需解压重压缩，避免文件损坏和闪退问题。

【附赠测试模型】
作者十二特别提供了一只名为 "Kipfel" 的小模型（送你一只小模型.obj），
可用于测试 OBJ 转 Mesh 功能，帮助新手快速上手打包流程。

│ "inCloset": true      │ "inCloset": true  │ 保持衣柜中可见      │

【完整性绕过原理】

修改 libBootloader.so 文件：

· 查找所有 "integrity" 字符串（共9个字符）
· 将其替换为 9 个空字节（\x00）
· 使游戏启动时跳过完整性检查

※ 完整性绕过检查.py 即为此功能的独立版本

================================================================================
第十章 快速上手指南
================================================================================

【场景一：首次使用，想转换一个模型】

1. 安装环境（参考第一章）
2. 将 .mesh 文件放在脚本同目录
3. 运行：python 下载这个就行.py
4. 选择文件，使用 hybrid 模式
5. 得到 .obj 文件，用 Blender 打开查看

【场景二：想制作 Mod，修改模型后打包】

1. 用 Blender 修改模型，导出为 .obj
2. 准备一个同版本的原始 .mesh 作为模板
3. 运行：python 自定义mesh半成功加解mesh.py
4. 选择菜单项 4（OBJ 转 Mesh）
5. 输入 .obj 路径和模板路径
6. 得到 repacked.mesh，放入游戏测试

【场景三：想解锁全物品】

1. 将 APK 文件放在脚本同目录
2. 运行：python 全物品+地图修改.py 或 python 全物品修改.py
3. 选择功能
4. 等待完成，重新签名后安装

【场景四：只想绕过完整性检查】

1. 将 APK 文件放在脚本同目录
2. 运行：python 完整性绕过检查.py
3. 选择 APK 文件并确认
4. 等待完成，重新签名后安装

【场景五：测试 OBJ 转 Mesh 功能】

1. 使用附赠的测试模型：送你一只小模型.obj（Kipfel）
2. 准备一个同版本的 .mesh 模板
3. 运行打包脚本，选择菜单项 4
4. 验证生成的 .mesh 文件是否正常
